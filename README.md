# Eclipse Trajectory Explorer

An interactive visualization of every total solar eclipse path on Earth from 1600 to 2100.

Built with:
- D3.js
- Vite
- TopoJSON

Features:
- Explore more than 500 years of eclipse trajectories.
- Highlight upcoming eclipses visible from Spain.
- Interactive tooltips.
- Maximum eclipse locations.
- NASA / GSFC eclipse geometries by Fred Espenak.

Data:
NASA GSFC / Fred Espenak
https://eclipsewise.com/
